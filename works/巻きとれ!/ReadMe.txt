【ゲーム内容】
親方
「遊び方は簡単だ！　ただひたすらに紙を巻き取るのみ！
　だが、紙の耐久には気をつけろよ。		　　」

【実行方法】
フォルダ内のpaper.exeを起動する。


【動作環境】
・OS       Microsoft Windows 98/2000/XP/Vista/7
・CPU　　  Intel(R) Core(TM) i5-3317U 1.70GHz
・メモリ   128MB以上
・HDD      1GB以上     
・DirectX9以降がインストールされている

【操作方法】
　・左シフトキー   	：ボタン１
　・右シフトキー	：ボタン２
　・スペースキー	：スタートボタン

セレクト画面での操作
・↑,↓ボタン　 ：項目の選択
・スタートボタン：決定

ランキング画面での操作
・スタートボタン：タイトルに戻る

ゲーム画面での操作
・ゲーム中
　・ボタン１　 ：左手を動かす
　・ボタン２　 ：右手を動かす
